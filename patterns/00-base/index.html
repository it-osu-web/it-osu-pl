IT@OSU PatternLab

This is the official design library for IT@OSU.

It is structured following the principles of the Atomic Design Theory:

  • Base elements
  • Atoms
  • Molecules
  • Organisms
  • Templates
  • Pages

For more information, consult our github repo.
