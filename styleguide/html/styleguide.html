    buttons

    ▼

    Button Usage

    Add the button class to apply the default button style. Add an additional hollow class to create a hollow (inverse) button.

    Use the <button> tag if the button performs an action that changes something on the current page or submits a form. Add the attribute type="button" to <button> elements, unless the button submits a form.

    Use the <a> tag if the "button" is a link to another page, or a link to an anchor within a page.

    Colors

    Add color classes to give buttons additional meaning.

    • primary (default color)
    • secondary
    • success
    • alert
    • warning
    • disabled (must also add aria-disabled="true" to button attributes when using the disabled variant)
    Sizes

    Add size classes to change the button's size. Omit a size class to use the default size.

    • small
    • large
